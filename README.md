# frame_miner
### 目标帧挖掘机
A focused data collection tool designed to extract useful frames from video footage. It allows users to manually capture and classify images into specific folders (e.g., class_A, class_B) in real-time, creating a clean, organized source for training datasets.

一款专注的数据收集工具，旨在从视频素材中提取有用的帧。它允许用户实时手动截取并将图片分类到特定文件夹（如 class_A, class_B），为训练数据集创建一个干净、有序的源文件。
